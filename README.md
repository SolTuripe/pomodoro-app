# Pomodoro App ⏱

For this project, I followed this tutorial: https://freshman.tech/pomodoro-timer/ \
I love using a pomodoro app to stay focused and work in segments.

## Project Setup 📌

```
git clone https://github.com/SolTuripe/pomodoro-app.git
cd pomodoro-app
npm install
```
